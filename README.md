# MatchUp — Where you belong

Vietnam's pickleball platform: find groups, join tournaments, book courts.

## Stack
- React + TypeScript + Vite
- Tailwind + shadcn/ui
- Supabase (auth, RLS, realtime)
- react-i18next (en, vi, zh, th, id)
- Vercel deployment

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
